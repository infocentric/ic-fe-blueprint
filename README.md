# ic-blueprint

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
### starts an express-server on port 3000 and webpack on 8080
```
yarn run dev
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
